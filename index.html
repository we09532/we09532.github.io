Snail Net

Welcome to SnailNet – the Internet for Snails! SnailNet is a Scratch interface-based project built using Snail IDE, letting users publish their own websites using SNAP servers, GitHub servers, or their own custom servers. Originally started by Dogstudios55 (@Randomuser12342), I helped bring the website version to life. Click the plus to make a website! (Note: The editor currently supports only one file, but you can still code JS inside HTML.) Search a URL or maybe even build a search engine for SnailNet!

WECoin

WE Coin is a fun currency simulation project built on PenguinMod, using a Scratch interface. It lets users send pretend currency to each other, making it a playful way to explore digital transactions and community interaction. Try sending some WE Coins to your friends!
